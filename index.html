Project Development Notes

Personal progress, plans, and version history

This page contains development notes related to the GameMatcher project. It provides an overview of the current progress, planned improvements, and version history as part of ongoing development and evaluation.

Current Status

  • The website functions as a static web application that allows users to browse and search game information based on preferences such as genre, mood, and other attributes.
  • Core pages such as Home, About, Contact, Sites, and Game Detail have been implemented and are accessible through the navigation bar.
  • Game data is stored in JSON format and managed externally using Google Spreadsheet, then loaded into the website for display.
  • Hosting is handled via Firebase, enabling public access to the website.
  • Basic search and filter features are available to help users find suitable games more efficiently.

Next Targets

  • Expanding the game database by adding more game entries and attributes to improve recommendation coverage. WIP
  • Improving data consistency and structure to support future feature expansion. WIP
  • Exploring the possibility of transitioning from a fully static website to a more dynamic system, including:
    - User accounts
    - Database-based content management
    - Easier editing and updating of game data without re-uploading the entire website
  • Enhancing user experience through interface refinements and clearer information presentation.
